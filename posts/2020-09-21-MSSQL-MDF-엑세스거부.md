## [SQL Server / MS-SQL] 운영 체제 오류 5: "5(액세스가 거부되었습니다.)"

MSSQL 서버 mdf 파일의 경로를 변경하여 Attach 할 경우 파일 접근권한이 없어서 발생하는 오류의 경우, 해당 폴더에 
NT Service\{SQL server INSTANSCE} 의 권한을 부여하여야 한다.

ex) NT Service\MSSQLSERVER
